# Rapid CLI

## Prerequisites

-   Install [node.js](https://nodejs.org/en) and [rust](https://www.rust-lang.org/)
-   Install your favorite package manager (npm, pnpm, yarn)

<br />
## Installation
<hr />

### Install the rapid cli with cargo:

<CodeBlock language='sh' code={`cargo install rapid-cli`} />

<br />
## Usage
<hr />
{/* TODO: in the future, provide heading links to relevant sections here*/}
The rapid framework is versatile. You choose how much of it you want to use.

<br />

-   Scaffold a new [fullstack rapid project](https://github.com/Cincinnati-Ventures/rapid/tree/main/examples/fullstack)
-   Integrate Rapid's frontend or backend with your existing application

<br />
### Generate a new fullstack rapid project:
<CodeBlock language='sh' code={`rapid new`} />

<br />
### Integrate the rapid backend into your existing application:
<CodeBlock language='sh' code={`rapid init server`} />

<br />
### Integrate the rapid frontend into your application:
<CodeBlock language='sh' code={`npm install @rapid-web/ui`} />

<br />
## Start your application:
<hr />
<br />
### Run the backend:
<CodeBlock language='sh' code={`rapid run `} />
<br />
### Run the frontend:
<CodeBlock language='sh' code={`npm run dev`} />
