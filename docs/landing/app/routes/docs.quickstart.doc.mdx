## Prerequisites
<hr />

Install [node.js](https://nodejs.org/en), [rust](https://www.rust-lang.org/), and your favorite package manager

<br />
<br />
## Installation
<hr />

### Install the rapid cli with cargo:

<CodeBlock language='sh' code={`cargo install rapid-cli`} />

<br />
<br />
## Getting Started With Remix

<hr />

### Generate a new [fullstack Rapid remix project:](https://github.com/Cincinnati-Ventures/rapid/tree/main/examples/fullstack)

<CodeBlock language='sh' code={`rapid new --remix `} />

<br />
### Integrate the rapid backend into your existing application:
<CodeBlock language='sh' code={`rapid new --server`} />

<br />
### Integrate the rapid frontend into your application:
<CodeBlock language='sh' code={`npm install @rapid-web/ui`} />

<br />
<br />
## Getting Started With NextJS

<hr />

### Generate a new fullstack Nextjs project:

<CodeBlock language='sh' code={`rapid new --nextjs`} />

<br />
### Integrate the rapid backend into your existing application:
<CodeBlock language='sh' code={`rapid new --server`} />

<br />
### Integrate the rapid frontend into your application:
<CodeBlock language='sh' code={`npm install @rapid-web/ui`} />

<br />
<br />
## Start your application:
<hr />

### Run the backend:

<CodeBlock language='sh' code={`rapid run `} />

<br />
### Run the frontend:
<CodeBlock language='sh' code={`npm run dev`} />
